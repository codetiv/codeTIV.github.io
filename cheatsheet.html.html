It's codeTIV Cheatsheet

HTML

<article>: A section for content usually forum post, magazine, articles, etc.

<link>: provides link to your resources

<div>: none semantic section

<head>: tells the browser page information

<title>: provides a title on broswer tab

CSS

color: changes the color of the foreground or the characters.

background: style change to the background

padding: provides a "cushion" and alignment for your content

border: provides link to your resources

font-family: gives certain types of font styles, will need to link special font in hmtl document.
